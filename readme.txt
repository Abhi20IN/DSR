This repository contains web scraping scripts written in both Python & R programming languages
When using Python, I will be experimenting with Scrapy Framework as well as libraries such as Beautiful Soup and urllib2 and
When using R, I will be experimenting with libraries such as rvest etc.
Some important information is as follows;
 Environment
 OS: Windows 7 64 bit
 Python editor: Pycharm community edition
 Python version: 2.7

 To install scrapy framework use the command pip install Scrapy on command prompt.
 Additionally, if you are a windows user then read the file "scrapy_install_guide.pdf"
